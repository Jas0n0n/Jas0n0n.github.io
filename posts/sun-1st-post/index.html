Hello Hugo

​ My Name is Jason. I’m a Coder,Nice to meet you!

1. Hugo Basic

​ https://gohugo.io/getting-started/quick-start/

$ hugo new site quickstart
$ cd quickstart
$ git clone https://github.com/dillonzq/LoveIt.git themes/LoveIt

Local Run

$ hugo new posts/first_post.md
$ hugo new posts/first_post.md
$ echo theme = \"LoveIt\" >> config.toml
$ hugo serve -D
$ hugo --theme=LoveIt --baseUrl="https://Jas0n0n.github.io/"

2.Github Page

  1. https://pages.github.com/
  2. Page e.g. https://Jas0n0n.github.io/
  3. git clone https://github.com/Jas0n0n/Jas0n0n.github.io

3.First Commit

$ hugo --destination ./docs --buildDrafts --cleanDestinationDir
$ cp -rf quickstart/docs/* Jas0n0n.github.io/
$ cd Jas0n0n.github.io
git add .
git commit -m "init the website"
git push

4. Github Page Setting

  1. https://github.com/{Your Github ID}/{Your Github ID}.github.io/settings/pages
  2. General-Pages-Source,Branch choose Master,folder choose /root

5.ByPassing jekyll

$ cd quickstart/docs
$ touch .nojekyll  
$ cd Jas0n0n.github.io 
$ touch .nojekyll  
$ cp -rf quickstart/docs/* Jas0n0n.github.io/ 
$ git add .   
$ git commit -m "byPassing the jekyll process"  
$ git push

6.Update Article

$ hugo new posts/second_post.md
$ hugo --destination ./docs --buildDrafts --cleanDestinationDir
$ cp -rf quickstart/docs/* Jas0n0n.github.io/ 
$ git add .   
$ git commit -m "byPassing the jekyll process"  
$ git push
